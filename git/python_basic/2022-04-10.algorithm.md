#### 백준 알고리즘 
- 반복문



```python
#2739번
# 구구단
# N을 입력받은 뒤, 구구단 N단을 출력하는 프로그램을 작성하시오.
# 출력 형식에 맞춰서 출력하면 된다.

n = int(input())

for i in range(1,10):  # 1~9까지의 범위
    print(n, '*', i, '=', n*i)
```
