```python
# 백준 알고리즘
#2742번 기찍N
# 자연수 N이 주어졌을 때, N부터 1까지 한 줄에 하나씩 출력하는 프로그램을 작성하시오.
# 첫째 줄에 100,000보다 작거나 같은 자연수 N이 주어진다.
# 첫째 줄부터 N번째 줄 까지 차례대로 출력한다.

n = int(input())

for i in range(n,0,-1):
    print(i)
    
#range함수에서 역순으로 값을 출력하려면 (초기값,종료값,증가값)을 (n,0,-1)의 값을 넣어 출력한다
    

```
